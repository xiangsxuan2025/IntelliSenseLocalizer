VS .NET IntelliSense 中英文生成下载器 | a tool for generate .net Localized IntelliSense

简介

.NET6之后，智能感知.NET IntelliSense files文件微软不再发布。

但是，微软在线文档仍有本地化翻译。我们可以爬下来网页上的文字，自行生成 .NET IntelliSense files文件。

使用方法

1. 简体中文用户

简中翻译包@stratosblue同学已经爬取好了，可以运行本程序直接更新。

然后，就可以快捷编码啦！

2. 其他用户（自行构建、非简体中文用户）

请参考How to use脚本说明，运行爬取程序、更新。
